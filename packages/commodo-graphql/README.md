A collection of frequently used GraphQL resolvers, 
to be used with [Commodo (@webiny/commodo)](../commodo) package.

## Install
```
npm install --save @webiny/commodo-graphql
```

Or if you prefer yarn: 
```
yarn add @webiny/commodo-graphql
```
